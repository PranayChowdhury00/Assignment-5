Important FAQ ?

what is the Document Object Model(DOM)

The Document Object Model (DOM) is a cross-platform and language-independent interface that treats an HTML or XML document as a tree structure wherein each node is an object representing a part of the document.

How do you select an element from the dom?

To select an element from the DOM (Document Object Model) in JavaScript, you use methods like getElementById() to access an element by its unique ID, getElementsByClassName() to find elements with a specific class name, or querySelector() to select an element using a CSS selector that can combine ID, class, tag name, and attributes for more complex targeting.

how do you manipulate an element attributes and style using dom?

To manipulate an element's attributes and style using the DOM in JavaScript, you first need to select the desired element using methods like getElementById, querySelector, or similar, then access its attributes using setAttribute to modify them and use the style property to directly change its CSS styles inline.

what is event delegation in the context of the dom, and why is it useful?

Event delegation refers to the process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated.
